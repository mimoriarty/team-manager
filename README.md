# team-manager

Web application to manage sports team rosters and lineups, schedule events (trainings, games, tournaments), and store results.

This repo contains a Gatsby + React frontend under `web/`. Jest is used for unit tests and Cypress is used for end-to-end tests.

Quick start (root commands assume Node.js >=18 and npm):

1. Install dependencies
   - cd web
   - npm install

2. Run development server
   - npm run develop

3. Run unit tests
   - npm test

4. Run Cypress e2e tests (interactive)
   - npm run cypress:open

5. Run Cypress e2e tests (headless)
   - npm run cypress:run
