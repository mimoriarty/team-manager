# team-manager

Web application to manage sports team rosters and lineups, schedule events (trainings, games, tournaments), and store results.

This repo contains a Vite + React application. Jest is used for unit tests and Cypress is used for end-to-end tests.

Backend will be a serverless supabase postgre database.
